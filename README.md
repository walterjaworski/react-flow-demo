## üá∫üá∏ About the project

This project showcases the creation of an **interactive visual flow** using
React Flow.

A *flow* is a structure composed of blocks (*nodes*) connected by links (*edges*).
This type of visualization is commonly used to represent:

- Processes
- Decision trees
- System integrations
- Business rules
- Conditional workflows

Each node represents a step in the process, while edges define the possible paths
between them.

The demo was built as a hands-on learning exercise. Throughout development, the
codebase evolved gradually, with a strong focus on:

- Dynamic node creation
- Conditional connections
- Automatic layout organization
- Clear separation of concerns (components, dialogs, forms, utils, types)
- Strong typing with TypeScript
- Commit history following **Conventional Commits**

Rather than focusing only on the final output, this project highlights **the
engineering process behind learning, adapting, and refining a solution**.
